== Description ==

Plugin creates language selector widget with country flags for the Stella plugin.
Plugin can be easly styled by css attached to your wordpress template.

== Installation ==

1. Upload `stella-flags.php` and folders from archive to the `/wp-content/plugins/stella-flags/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add new widget to the sidebar.

== Frequently asked questions ==

= How to add/change flag images =

You can add/change flag images in stella-flags/images folder. COUNTRY_NAME.gif for active flag and COUNTRY_NAME_off.gif fot the inactive flag. COUNTRY_NAME should be the same as in Stella configuration.
